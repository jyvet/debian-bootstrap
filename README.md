Debian bootstrap
================

Ansible playbook for configuring a fresh desktop environnement after minimal
debian installation.



Install
-------

Launch with root account:

    wget -O - -o /dev/null  https://raw.githubusercontent.com/jyvet/debian-bootstrap/master/run.sh | bash
